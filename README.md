# Toolkit

Small collection of browser tools for everyday web asset work. It is made for quick jobs like preparing a favicon, converting an SVG, or getting files ready before adding them to a website or app.

The tools are meant to be simple: open a tool, drop in a file, adjust the settings, and download the result. Everything runs in your browser, so your files do not need to be uploaded somewhere else while you work.

## What is inside

- Image to Favicon: turn an image into `favicon.ico` and a favicon pack.
- SVG Converter: turn SVG files into PNG or JPG.

## Run it locally

Install and start:

```bash
pnpm install
pnpm dev
```

Build:

```bash
pnpm build
```
